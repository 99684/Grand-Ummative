## Requirements:

> #### For Linux **(Debian only)**:
> - Python3 must be installed
> - Pip must be installed
> - Must support .sh files
> - Must have root permissions
> #### For Windows:
> - Python3 must be installed
> - Pip must be installed
> - Must support batchfile execution
> - Must be Windows 11
> - *Also supports windows 10 build 17063*
